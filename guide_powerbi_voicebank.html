Guide complet Power BI connecté à PostgreSQL VoiceBank

Avant tout, installe le driver PostgreSQL pour Power BI si ce n'est pas déjà fait.

Prérequis : Télécharge et installe le driver Npgsql depuis :
https://github.com/npgsql/npgsql/releases
Cherche le fichier Npgsql-x.x.x.msi → installe → redémarre Power BI
1

Ouvre Power BI Desktop

Clique sur Obtenir les données → cherche PostgreSQL

2

Remplis les champs de connexion

Serveur localhost
Base de données voicebank_db
Mode Importation
3

Identifiants

Utilisateur voicebank_user
Mot de passe voicebank2024

Une fois connecté, le navigateur de tables s'ouvre. Sélectionne ces éléments dans le schéma voicebank :

Tables à importer :

clients

45 211 lignes

transactions

1 296 675 lignes

credits

20 000 lignes

Vues à importer (déjà agrégées → rapides) :

v_dashboard_global

v_clients_par_banque

v_transactions_mensuelles

v_portefeuille_credit

Clique sur Transformer les données (pas Charger directement) pour vérifier les types avant import.

Dans l'onglet Modèle de Power BI, crée les relations entre les tables :

clients

id_client (PK)

transactions

id_client (FK) → 1:N

credits

id_client (FK) → 1:N

Pour créer une relation :

Onglet Modèle → glisse clients.id_client vers transactions.id_client → Power BI crée la relation automatiquement → répète pour credits

Voici les 6 visuels clés à créer pour le dashboard VoiceBank :

KPI Cards

Total clients · Volume transactions · Taux fraude · Encours crédit

Source : v_dashboard_global

Courbe transactions / mois

Volume mensuel avec courbe de fraudes superposée

Source : v_transactions_mensuelles

Répartition clients

Camembert Local vs Diaspora par pays de résidence

Source : clients

Carte géographique

Bulles sur carte par ville — volume transactions

Source : transactions

Fraudes par banque

Barres comparatives taux de fraude par banque

Source : v_clients_par_banque

Portefeuille crédit

Statuts crédits · Taux défaut · Encours par type

Source : v_portefeuille_credit

Ajoute des Slicers (filtres interactifs) pour : Banque · Ville · Période · Pays de résidence — ça rend le dashboard vraiment dynamique.

Une fois le dashboard terminé, publie-le sur Power BI Service pour obtenir un lien URL à intégrer dans VoiceBank :

1

Fichier → Publier → Power BI Service

Connexion avec ton compte Microsoft gratuit (compte étudiant parfait)

2

Récupère le lien du rapport

Sur Power BI Service → Fichier → Incorporer le rapport → Site web ou portail → copie l'URL

3

Ce lien → bouton dans VoiceBank

Dans l'interface React, le bouton ouvrira directement ce rapport :

window.open("https://app.powerbi.com/...", "_blank")
Dashboard Power BI publié + bouton intégré → Phase 2 complète ! On passe aux modèles IA 🚀
